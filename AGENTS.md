# Frontend Interview Knowledge Base

For requests that add interview questions, conduct mock interviews, evaluate answers,
review weaknesses, or randomly select a practice question, read and follow
`skills/interview-coach/SKILL.md` before responding.

Treat `knowledge-base/state.json` as the canonical record. Keep its generated Markdown
views synchronized through `skills/interview-coach/scripts/kb.py`. Do not invent prior
scores or answers when the knowledge base does not contain them.
